# AdSecureCheck - Audit de Sécurité Active Directory

## Description
Solution automatisée d'audit de sécurité pour Active Directory.

## Stack Technique
- **Backend**: Python 3.11+ / Flask
- **Frontend**: React.js / Nginx
- **Database**: PostgreSQL
- **Containerisation**: Docker / Docker Compose
- **Automatisation**: Ansible

## Installation

### Prérequis
- Docker & Docker Compose
- Python 3.11+
- Node.js 20+
- Ansible

## Structure du Projet
```
.
├── backend/          # API Flask + scripts d'audit
├── frontend/         # Interface React
├── ansible/          # Playbooks de déploiement
├── docs/            # Documentation
├── scripts/         # Scripts utilitaires
└── reports/         # Rapports générés
```
